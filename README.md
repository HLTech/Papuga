# Papuga
Papuga (/papu…°a/) is a mocking framework for unit tests which brings experience similar to [Mockito](https://github.com/mockito/mockito) and [SwiftyMocky](https://github.com/MakeAWishFoundation/SwiftyMocky). Framework uses [Sourcery](https://github.com/krzysztofzablocki/Sourcery) which scans swift files and generates mock classes for you.

## Installation
Papuga is available through [CocoaPods](https://cocoapods.org). To install it, simply add the following line to your Podfile:

```ruby
pod 'Papuga'
```

## Example
To run the example project, clone the repo and run `pod install` from the Example directory first or just run `pod try Papuga`.
